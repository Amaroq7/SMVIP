# VIP plugin for CSGO

# Features
- vip prefix in chat (cvar)
- weapon menu with own skins (cvar)
- greetings when a vip comes to a server
- dynamic motd
- free taser (cvar)
- additional hp at round start (cvar)
- free defuser (cvar)
- additional money at round start (cvar)
